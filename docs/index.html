vaults-diagram-tools

Portable Mermaid and source-code diagram toolkit for SVG/JPEG rendering, offline assets, and MCP workflows.

Mermaid to SVG/JPEG Source diagrams MCP stdio Offline vendor runtime 
npm install -D vaults-diagram-tools
npx vaults-mermaid-render diagram.mmd --output-dir out
npx vaults-source-diagrams --source-dir src --output-dir diagrams
npx vaults-diagram-mcp

Download / Install

npm registry

npm install -D vaults-diagram-tools

One-shot npx

npx --yes --package vaults-diagram-tools vaults-mermaid-render diagram.mmd --output-dir out

GitHub package

npm install github:malnati/vaults-diagram-tools

Container

docker run --rm ghcr.io/malnati/vaults-diagram-tools:v0.1.1 --help

Release files

npm tarball · zip archive

Renderer

vaults-mermaid-render renders Mermaid source to SVG/JPEG, manifests, and optional PNG or text sidecars.

Source diagrams

vaults-source-diagrams creates Mermaid diagrams from source-code structure without Vault-specific paths.

MCP

vaults-diagram-mcp exposes render_mermaid_text, render_mermaid_file, and generate_source_diagrams.

CDN facade

CDN endpoints expose browser-safe helpers only. Rendering remains Node, container, or MCP side.
