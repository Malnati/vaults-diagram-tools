vaults-diagram-tools

Portable Mermaid and source-code diagram toolkit for SVG/JPEG rendering, offline assets, and MCP workflows.

npm install github:malnati/vaults-diagram-tools
vaults-mermaid-render diagram.mmd --output-dir out --png
vaults-source-diagrams --source-dir src --output-dir diagrams
vaults-diagram-mcp

Renderer

Render Mermaid source to SVG/JPEG, manifests, and optional ASCII/Unicode sidecars.

Source diagrams

Generate Mermaid diagrams from source-code structure without Vault-specific paths.

MCP

Expose three explicit tools for rendering text, rendering files, and generating source diagrams.
